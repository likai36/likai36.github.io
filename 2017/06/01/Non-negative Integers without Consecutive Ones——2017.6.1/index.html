Non-negative Integers without Consecutive Ones

发表于 2017-06-01 | 分类于 算法设计 |

The Question

Given a positive integer n, find the number of non-negative integers less than or equal to n, whose binary representations do NOT contain

The Example

Input: 5
Output: 5
Explanation:
Here are the non-negative integers <= 5 with their corresponding binary representations:
0 : 0
1 : 1
2 : 10
3 : 11
4 : 100
5 : 101
Among them, only integer 3 disobeys the rule (two consecutive ones) and the other 5 satisfy the rule.

The Anwser

从 n-1 -> n 如何append？
假设f[n]为n位情况下，非连续1的个数，
则：
f[n] = “0”f[n-1] + “1”f[n-1];
分出一位，我们构建递归式，可以是0，也可以是1，所以有两种情况，但上述递归式有问题。
主要在于：”1”f[n-1]，如果f[n-1]是非连续1的个数，而f[n-1]的解，并没有约束开头不能是”1”，所以
“1”f[n-1] = “10”f[n-2] + “11”f[n-2];
而”11”f[n-2]是非法的，所以：
f[n] = “0”f[n-1] + “10”f[n-2];
得：
f[n] = f[n-1] + f[n-2];
这样，我们可以求得任何位数的非连续1的个数。
