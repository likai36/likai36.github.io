发表于 2017-03-05 | 分类于 算法设计 |

The Question

Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

The Anwser

题目的意思表达的比较清楚，k个排好序的链表，将其重组为一个排序的链表。

解法1（逐一合并）

  • k个链表顺序取两个链表进行排序；
  • 排好序的链表和下一个接着排序；
  • 对链表进行排序时需要判断几种情况，一种是一个链表为空，一个不为空；或者两个都为空、不为空；
  • 当都不为空的时候，需要排序插入新链表之中，但是需要注意结束条件，当排序过程中一个为空，另外一个就可以全部插入进入新链表之中了。
