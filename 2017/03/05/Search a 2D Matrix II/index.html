Search a 2D Matrix II

发表于 2017-03-05 | 分类于 算法设计

The Question

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

  • Integers in each row are sorted in ascending from left to right.
  • Integers in each column are sorted in ascending from top to bottom.

The Example

Consider the following matrix:

[[1, 4, 7, 11, 15],
[2, 5, 8, 12, 19],
[3, 6, 9, 16, 22],
[10, 13, 14, 17, 24],
[18, 21, 23, 26, 30]]

Given target = 5, return true。Given target = 20, return false.

The Anwser

题目的意思表达的比较清楚，查看矩阵中是否存在一个数字，存在返回true，没有就返回flase。

我的思路

  • 和第一行的最后一个数字进行比较；
  • 比它小，就降低列数；
  • 比它大，增大行数；
  • 等于它，就返回Treu；
  • 找完也没有就返回False。

  • 中心数字和target相等的话，那么就可以返回true；
  • 中心数字大于target的话，说明target如果存在只可能存在于左上三块之中；
  • 中心数字小于taeget的话，说明target如果存在只可能存在于右下三块之中；
  • 直到划分的矩阵变为只有一个数字的情况，如果这个时候还是不相等，那么矩阵中就不存在target。
