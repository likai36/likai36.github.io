Majority Element

发表于 2017-03-03 | 分类于 算法设计 |

The Question

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.You may assume that the array is non-empty and the majority element always exist in the array.

The Anwser

本题的难度不是很大，主要有一下几种解法。

  • 计算vector中每个元素的出现次数，出现次数最大的即为要求的Majority Element，可以使用hash表。map或者unordered_map。
  • 将vector排序，拍完序之后，中间的数字即为Majority Element。排序函数可以为sort（）或者nth_element（），但是要注意nth_element（）函数的参数意义。
  • 暴力的还可以使用随机数的方法，因为有二分之的以上的概率取得的一个值即为Majority Element。就有理由说有很大几率直接取值就是要求的数据。可以使用srand()和rand()。
  • 分治方法。划分vector，每一次求小的vector的Majority Element

3.概率取值，并验证是否成立

srand()就是给rand()提供种子seed。如果srand每次输入的数值是一样的，那么每次运行产生的随机数也是一样的， 通常的做法是 以这样一句代码srand((unsigned) time(NULL));来取代，这样将使得种子为一个不固定的数， 这样产生的随机数就不会每次执行都一样了。
