Different Ways to Add Parentheses

发表于 2017-03-07 | 分类于 算法设计

The Question

Given a string of numbers and operators, return all possible results from computing all the different possible ways to group numbers and operators. The valid operators are +, - and *.

The Example

Example 1

Input: “2-1-1”。Output: [0, 2]

  • ((2-1)-1) = 0
  • (2-(1-1)) = 2

Example 2

Input: “23-45”。Output: [-34, -14, -10, -10, 10]

  • (2(3-(45))) = -34
  • ((23)-(45)) = -14
  • ((2(3-4))5) = -10
  • (2((3-4)5)) = -10
  • (((23)-4)5) = 10

The Answer

对于本题，思路还是比较简单的，由于string之中只有两种类型的字符：数字和操作符。因此对于string的转化为数学表达式的计算结果而言，就是一个操作符的排列问题。具体可以如下操作：

  • 顺序遍历string直到遇到一个操作符；
  • 以操作符为区分，操作符左侧的为一个排列问题，位于右侧的是一个排列问题；
  • 对于左右两侧的的排列问题，就是一个递归的操作，两侧的取值分别存储在两个vector之中；
  • 对于当前的特定的操作符，string的计算结果就是左右两个vector内存储的数值的组合问题。
