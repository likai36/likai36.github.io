Binary Tree Paths

发表于 2017-03-09 | 分类于 算法设计 |

The Question

Given a binary tree, return all root-to-leaf paths.

The Example

For example, given the following binary tree：1,2,3,NULL,5
All root-to-leaf paths are：[“1->2->5”, “1->3”]

The Answer

本题的思路比较简单，就是从root节点进行判断，左右顺序遍历，直到遍历到叶节点时刻，将遍历的路径path插入到一个vector之中，所有情况都完成返回这个vector即可。

注意点

虽然本题思路比较简单，但是实现起来还是有很多坑的，我就跳进去一个好久没有出来。几个我遇到的坑分享如下：

  1. 递归函数不能定义为vector<string>的形式，因为这种情况就需要分情况进行判断，当出现遍历到的节点左右子节点都不为空的时刻就会出现问题。
  2. 递归函数应该设置为void，无返回值类型的函数，这个时候就需要找一个参数可以全局传递的。
  3. 设置全局传递的参数vector<string>的时候，需要设置为引用的形式，参数传递需为按索引传递。不然传递进来的就是形参，为按值传递。
