Longest Palindromic Subsequence

发表于 2017-05-05 | 分类于 算法设计 |

The Question

Given a string s, find the longest palindromic subsequence’s length in s. You may assume that the maximum length of s is 1000.

The Example

Example1

Input:”bbbab”
Output:4
One possible longest palindromic subsequence is “bbbb”.

Example 2:

Input:”cbbd”
Output:2
One possible longest palindromic subsequence is “bb”.

The Anwser

使用动态规划的求解方式。
使用动态规划最重要的就是明确这个大问题如何用子问题来表示，然后分情况写出动态规划的递归表达式。

子问题表达

给定一个字符串，把它划分成什么样的子问题呢？通过分析我们发现，这样的子问题是：任意一个该字符串的子串包含的回文串的长度，那么对于这个字符串的子串，我们可以用它在原字符串中的起始位置和结束位置来表示。因此，我们可以用一个二维数组来对子问题进行表示，dp[i][j]表示the longest palindromic subsequence’s length of substring(i, j)。

问题分解行

Initialization: dp[i][i] = 1，dp[i][j]={0}
if s[.charAt(i)[i] == s.[j]：dp[i][j] = dp[i+1][j-1] + 2
otherwise： dp[i][j] = Math.max(dp[i+1][j], dp[i][j-1])
