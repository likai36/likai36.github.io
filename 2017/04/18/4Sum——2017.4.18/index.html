4Sum

发表于 2017-04-18 | 分类于 算法设计 |

The Question

Given an array S of n integers, are there elements a, b, c, and d in S such that a + b + c + d = target? Find all unique quadruplets in the array which gives the sum of target.

Note: The solution set must not contain duplicate quadruplets.
就是给一个数组，在其中寻找所有的四个数字的组合，这四个数组字和为target。

The Example

The Anwser

使用递归求解的方式，结果是可以Accepted，但是：Time Limit Exceeded。

  1. 判断数组的长度，长度小于四就不用判断了
  2. 对数组进行递增排序
  3. 对数组从头开始判断所有的四个数字的组合，也就是path插入第一个其他的就可以递归的插入，在验证是不是target即可
  4. 对于符合条件的path，全部插入到结果中
  5. 对于结果进行去重复。

由于使用的是递归的方式，重复判断的情况很多，所以超时，使用如下方式：

  1. 对数组进行排序
  2. 确定四个数字中的前两个数字
  3. 对着两个数字之后的数字，设置left和right指示；
  4. 求这四个数字的和是否为target的需求；
  5. 当大于target的时候，right减小
  6. 当小于target的时候，left变大
  7. 循环判断所有的情况。
