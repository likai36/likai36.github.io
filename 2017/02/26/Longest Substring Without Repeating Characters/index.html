Longest Substring Without Repeating Characters

发表于 2017-02-26 | 分类于 算法设计 |

The Question

Given a string, find the length of the longest substring without repeating characters.

The Example

Given “abcabcbb“, the answer is “abc“, which the length is 3.
Given “bbbbb“, the answer is “b“, with the length of 1.
Given “pwwkew“, the answer is “wke“, with the length of 3. Note that the answer must be a substring, “pwke” is a subsequence

The Anwser

tips

  • 字符串长度为0时刻；
  • 字符串长度为1时刻；
  • 字符串长度大于1时刻。

我的思路

主要想法就是嵌套遍历，外层遍历字符串，内层遍历外层遍历过的字符串。这个时候有一个关键点就是，当内层j遍历到外层i处时刻也就是当s[i]=s[j]时刻也就是这个时刻字符串已经有重复现象出现了，那么我们就可以设置内层下次开始遍历的地方就是这个关键点的下一个位置，设置flag=j+1，并且结束内层遍历。只要没有出现这种情况，longest就可以一直增加了。之后每一个点比较一下上次的longest和本次temp的大小用阿里设定新的longest。
